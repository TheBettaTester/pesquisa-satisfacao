## 📌 Sobre o projeto

Este projeto é um sistema de pesquisa de satisfação desenvolvido em Python.

O objetivo é coletar e analisar a opinião de entrevistados sobre um determinado atendimento, classificando as respostas em diferentes níveis de satisfação.

---

## 🛠️ Tecnologias utilizadas

- Python  
- VS Code  
- GitHub  

---

## ⚙️ Funcionalidades

- Cadastro de entrevistados (nome e idade)
- Validação de dados (nome e idade)
- Classificação por faixa etária:
  - Criança
  - Adolescente
  - Adulto
  - Idoso
- Coleta de opinião:
  - Excelente
  - Bom
  - Ruim
- Contagem automática das respostas
- Exibição do resultado final da pesquisa
- Interface no terminal com cores (ANSI)

---

## 📊 Estrutura da pesquisa

O sistema realiza entrevistas com uma quantidade definida de participantes (por padrão, 50).

Para cada entrevistado, são coletados:
- Nome
- Idade
- Opinião sobre o atendimento

---

## 🧠 Lógica do programa

O programa utiliza:
- Estruturas de repetição (`for` e `while`)
- Validação de entrada de dados
- Contadores para armazenar resultados
- Condições (`if/elif`) para classificação
